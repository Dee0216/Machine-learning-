   └─ chmod +x setup.sh

═════════════════════════════════════════════════════════════════════════════

POST  /api/upload/model/           Upload H5 model

🐛 TROUBLESHOOTING:
═════════════════════════════════════════════════════════════════════════════

Problem: "Module not found"
Solution: pip install -r requirements.txt

Problem: Port 8000 in use
Solution: python manage.py runserver 8001

Problem: TensorFlow won't install
Solution: pip install tensorflow-cpu

Problem: Database errors
Solution: python manage.py migrate --run-syncdb

More help: See STARTUP_GUIDE.md
